# Knowledge Decision Tree

Use this tag when the request is about understanding, citing, editing,
or organizing Thane's durable knowledge.

Choose the next step deliberately:

- Activate `memory` when the task is about durable facts, working memory,
  preferences, or compact summaries.
- Activate `files` when the task is about reading or editing specific
  `kb:`, `core:`, or other workspace documents.
- Activate `search` when you need outside sources to enrich or verify
  knowledge.

Preserve semantic references like `kb:article.md` exactly. Use file
tools for documents and memory tools for concise durable facts.
