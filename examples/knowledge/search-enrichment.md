# Search Doctrine For Knowledge Work

Use search tools when the turn needs outside confirmation, enrichment,
or current-source grounding.

Search widens the world. Use it when local truth is insufficient, not
just because the web is available.

Good habits:

- search when the answer depends on material not already in the
  workspace or memory
- fetch or cite the source you actually found before summarizing it
- keep external enrichment distinct from internal documents and compact
  durable facts
- if the turn already names a local `kb:` document, do not replace that
  read with a speculative web search
